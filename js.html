Introduction to JavaScript

JavaScript is a versatile, high-level programming language used for adding interactivity, animations, and dynamic behavior to web pages. It runs directly in the browser.

Basic Syntax

JavaScript is embedded in HTML using the <script> tag. Example:


<script>
    alert("Hello, World!");
</script>

Variables and Data Types

JavaScript supports various data types like strings, numbers, booleans, arrays, and objects. Example:


let name = "Ade";
let age = 25;
let isDeveloper = true;

console.log(name, age, isDeveloper);

Control Structures

JavaScript supports conditional statements and loops for decision-making and iteration. Example:


if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

for (let i = 0; i < 5; i++) {
    console.log(i);
}

Advanced JavaScript Topics

1. DOM Manipulation

The Document Object Model (DOM) allows JavaScript to interact with HTML and CSS. Example:


document.getElementById("demo").innerHTML = "Hello, JavaScript!";

2. Event Handling

Reacting to user interactions like clicks, keyboard events, etc. Example:


document.querySelector("button").addEventListener("click", function() {
    alert("Button Clicked!");
});

3. Asynchronous JavaScript (Promises & Async/Await)

Handling asynchronous operations with Promises and Async/Await. Example:


async function fetchData() {
    try {
        let response = await fetch('https://api.example.com/data');
        let data = await response.json();
        console.log(data);
    } catch (error) {
        console.error('Error:', error);
    }
}

4. Object-Oriented Programming (OOP)

Creating objects, classes, and inheritance. Example:


class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    greet() {
        console.log(Hello, my name is ${this.name}.);
    }
}

const person1 = new Person("Ade", 25);
person1.greet();
