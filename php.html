Introduction to PHP

PHP (Hypertext Preprocessor) is a powerful server-side scripting language used for creating dynamic and interactive web pages.

PHP Syntax

PHP code is written inside <?php ... ?> tags. Example:


<?php
    echo "Hello, World!";
?>

Variables and Data Types

PHP supports various data types such as strings, integers, booleans, arrays, and objects. Example:


<?php
    $name = "Ade";
    $age = 25;
    $isDeveloper = true;

    echo $name . " is " . $age . " years old.";
?>

Control Structures

PHP supports if-else statements, loops, and switch cases for decision-making. Example:


<?php
    if ($age >= 18) {
        echo "You are an adult.";
    } else {
        echo "You are a minor.";
    }

    for ($i = 0; $i < 5; $i++) {
        echo $i;
    }
?>

Advanced PHP Topics

1. Object-Oriented Programming (OOP)

PHP supports OOP, allowing you to create classes, objects, inheritance, and polymorphism. Example:


<?php
    class Person {
        public $name;
        public $age;

        public function __construct($name, $age) {
            $this->name = $name;
            $this->age = $age;
        }

        public function greet() {
            return "Hello, my name is " . $this->name;
        }
    }

    $person1 = new Person("Ade", 25);
    echo $person1->greet();
?>

2. File Handling

PHP allows you to create, read, write, and delete files. Example:


<?php
    $file = fopen("test.txt", "w");
    fwrite($file, "This is a test file.");
    fclose($file);
?>

3. Sessions and Cookies

Sessions and cookies are used to store user data across multiple pages. Example:


<?php
    session_start();
    $_SESSION["username"] = "Ade";
    echo "Session saved successfully!";
?>

4. PDO & MySQLi (Database Handling)

PHP provides ways to interact with databases using PDO or MySQLi. Example:


<?php
    $conn = new PDO("mysql:host=localhost;dbname=testdb", "root", "");

    $stmt = $conn->prepare("SELECT * FROM users");
    $stmt->execute();

    $result = $stmt->fetchAll();
    foreach ($result as $row) {
        echo $row['name'];
    }
?>
