MySQL Tutorial - A_Tech

Introduction to MySQL

MySQL is an open-source relational database management system used to store and manage data efficiently.

Creating a Database

To create a database, use the CREATE DATABASE statement:


CREATE DATABASE myDatabase;

Creating Tables

To create a table, use the CREATE TABLE statement:


CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(50),
    email VARCHAR(50),
    age INT
);

Inserting Data

To insert data into a table, use the INSERT INTO statement:


INSERT INTO users (name, email, age) 
VALUES ('Ade', 'ade@gmail.com', 25);

Retrieving Data

To retrieve data, use the SELECT statement:


SELECT * FROM users;

Updating Data

To update data, use the UPDATE statement:


UPDATE users SET email = 'newemail@gmail.com' WHERE id = 1;

Deleting Data

To delete data, use the DELETE statement:


DELETE FROM users WHERE id = 1;

Advanced MySQL Topics

1. Joins

Joins are used to combine rows from two or more tables based on a related column.


SELECT users.name, orders.product
FROM users
INNER JOIN orders ON users.id = orders.user_id;

2. Indexing

Indexes speed up search queries:


CREATE INDEX idx_name ON users (name);

3. Stored Procedures

Stored procedures are precompiled SQL statements for complex queries:


DELIMITER //
CREATE PROCEDURE GetUsers()
BEGIN
    SELECT * FROM users;
END //
DELIMITER ;

4. Triggers

Triggers automatically perform tasks when certain actions occur:


CREATE TRIGGER before_insert_user
BEFORE INSERT ON users
FOR EACH ROW
SET NEW.name = UPPER(NEW.name);

5. Transactions

Transactions ensure multiple SQL operations execute as a unit:


START TRANSACTION;
INSERT INTO orders (user_id, product) VALUES (1, 'Laptop');
COMMIT;

6. User Privileges

Managing user access with permissions:


GRANT ALL PRIVILEGES ON myDatabase.* TO 'username'@'localhost';
